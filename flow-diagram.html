✦ PlanMyTravel — App Architecture

A complete flow diagram for frontend developers: execution order, routing, components, clicks, and data flow.

1 Execution Order — What Runs First

Every single page request — regardless of URL — starts at layout.tsx. It loads fonts, wraps the app in AuthSessionProvider (makes session available everywhere), then hands off to the matching route's page.tsx.
